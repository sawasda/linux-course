# H2

1. Asensin micro-editorin terminaalissa komennolla; sudo apt install micro
2. Asensin kolme komentoriviohjelmaa komennolla; sudo apt install htop mc tree
3. Ensin kokeilin htop komennolla asennettua komentoriviohjelmaa, joka näyttää järjestelmää reaaliaikaisesti, prosessit, CPU:n käytön, muistin käytön yms.
![Kuva](h2kuva9.png)
4. Sitten kokeilin mc komennolla asennettua komentoriviohjelmaa, joka on tiedostonhallintatyökalu tiedsotojen kopioimiseen, poistamiseen tai siirtämiseen.
  ![Kuva](h2kuva8.png)
5. Sitten kokeilin tree komennolla asennettua komentoriviohjelmaa, joka näyttää hakemistot "puunmallisena". Esimerkissä viime tunnilla luodut kansiot.
  ![Kuva](h2kuva7.png)
6. Kansioiden esittelyt.
a) Root hakemiston näyttämisen komennolla ls;
b) Kotihakemiston näyttämisen ls komennolla;
c) Käyttäjän työpöytä;
d) ls /etc/ esimerkkitiedosto; hostname
e) ls /media/ esimerkiksi cd-levy
f) ls /var/log/ esimerkkinä alternatives.log
  ![Kuva](h2kuva5.png)
  ![Kuva](h2kuva4.png)
7. Grep-komentoesimerkkejä. Grep-komennolla etsitään hakusanalla tiedostoja. Käytin komentoja grep "root" grep "error" ja grep "bash"
  ![Kuva](h2kuva3.png)
8. Pipe yhdistää kaksi komentoa. Oli vaikeuksia | -merkin tekemisessä ja käytin Unicode-koodin syöttöä; ctrl + shift + u & koodi 7c
Pipe -esimerkki; ls /home/sara/ | wc -l tulostaa montako riviä tuloksessa on;
  ![Kuva](h2kuva2.png)
9. Komento sudo lshw -short -sanitize;
  ![Kuva](h2kuva1.png)
